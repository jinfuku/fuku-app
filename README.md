## 如何使用 APP 原型

### 方法一：在电脑浏览器中查看

1. 打开文件资源管理器
2. 导航到：`c:/Users/jinfu/WorkBuddy/Claw/fuku-app/`
3. 双击 `index.html` 文件
4. 网页将在浏览器中打开

### 方法二：使用 Python HTTP 服务器

1. 打开命令行（PowerShell）
2. 进入 APP 目录：
   ```bash
   cd c:/Users/jinfu/WorkBuddy/Claw/fuku-app
   ```
3. 启动 Python 服务器：
   ```bash
   python -m http.server 8000
   ```
4. 在浏览器中打开：
   ```
   http://localhost:8000
   ```

### 方法三：在手机浏览器中查看

1. 确保电脑和手机在同一个 WiFi 网络
2. 在电脑上启动 Python HTTP 服务器（方法二）
3. 查看电脑的 IP 地址：
   - Windows: 打开命令行，输入 `ipconfig`
   - 找到 "IPv4 地址"，例如：192.168.1.100
4. 在手机浏览器中输入：
   ```
   http://192.168.1.100:8000
   ```
5. 将网页添加到主屏幕，创建桌面快捷方式

## 技术说明

### 为什么先用 HTML 原型？

1. **快速体验**：福库老师可以立即在浏览器中看到 APP 效果
2. **无需安装**：不需要安装复杂的开发工具
3. **易于修改**：HTML/CSS/JS 容易理解和修改
4. **跨平台**：任何设备都可以访问（电脑、手机、平板）

### 未来如何转换成原生 APP？

HTML 原型可以作为 Flutter APP 的 WebView 内容，或者使用框架（如 Capacitor、Ionic）直接打包成原生应用。
